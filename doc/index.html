COLORS
HTML
<button class= "btn btn--primary btn--large">PRIMARY</button>
Use primary buttons for main actions on a page.
HTML
<button class= "btn btn--secondary btn--large">SECONDARY</button>
Use secondary button if you need another action and you are already using primary button.
HTML
<button class= "btn btn--tertiary btn--large">TERTIARY</button>
Tertiary buttons are great alternative to primary and secondary buttons.
HTML
<button class= "btn btn--success btn--large">SUCCESS</button>
Success button is used to indicate the completion of any action successfully.
HTML
<button class= "btn btn--alert">ALERT</button>
Alert button is used to warn about something to the user.
